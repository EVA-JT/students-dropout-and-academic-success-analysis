Contexto da base de dados

A base de dados escolhida criada a partir de uma instituio de ensino superior (adquirido de diversas bases de dados distintas) relacionado a alunos matriculados em diferentes cursos de graduao, como agronomia, design, educao, enfermagem, jornalismo, administrao, servio social e tecnologias. O conjunto de dados original inclui informaes conhecidas no momento da matrcula do aluno (trajetria acadmica, dados demogrficos e fatores socioeconmicos) e o desempenho acadmico dos alunos ao final do primeiro e segundo semestres.

A base foi alterada para otimizar a eficincia da anlise. Foram removidas no total 23 colunas de informaes consideradas redundantes ou irrelevantes. Os nomes das colunas restantes, bem como os valores da varivel Resultado, foram traduzidos para o portugus, colunas que utilizavam codificao numrica para representar categorias (como a coluna Gnero, que usava 0 e 1) foram convertidas para seus respectivos rtulos descritivos (Homem e Mulher), tornando a base mais legvel.

calc_prop <- function(var_nome) {
  desistentes_sim <- sum(df_q1[[var_nome]] == "Sim")
  total_sim <- sum(df[[var_nome]] == "Sim")
  return(round(desistentes_sim / total_sim, 3))
}

# Aplicando a funo
prop_bolsista <- calc_prop("Bolsista")
prop_devedor <- calc_prop("Devedor")
prop_deslocado <- calc_prop("Deslocado")
prop_nees <- calc_prop("Necessidades.educacionais.especiais")

# Juntando em uma tabela
proporcoes <- c(Bolsista = prop_bolsista, Devedor = prop_devedor, Deslocado = prop_deslocado, Necessidades = prop_nees)

barplot(proporcoes, col = "steelblue", ylim = c(0, 1), main = "Proporo de Desistentes em cada Grupo", ylab = "Proporo", las = 1)

O grfico revela que os alunos devedores e aqueles com necessidades especiais apresentam as maiores taxas proporcionais de desistncia. Isso sugere que fatores financeiros e de acessibilidade podem estar fortemente associados evaso escolar.
